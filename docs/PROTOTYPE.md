**关键交互**：
- 点击"开始" → 申请麦克风权限 → 实时显示音高
- 拨动琴弦 → 音名快速更新 → 偏差条左右移动
- 偏差 < 5 cents → 整体变绿 + 振感反馈
- 长按目标弦 → 切换调弦法（标准/Drop/Baritone）

## 4. 交互细节规范

### 4.1 设计原则
- **简洁优先**：核心功能 3 步内可达
- **即时反馈**：所有操作 < 100ms 响应（除 AI 推理）
- **离线优先**：网络断开不破坏流程
- **无障碍**：所有交互支持 VoiceOver / TalkBack

### 4.2 颜色与字体（沿用现有 App 主题）
- 主色：薄荷绿 `#4ECDC4`
- 强调色：木色 `#D4A574`
- 错误：红 `#FF4D4F`
- 字体：系统默认（PingFang / SF Pro / Roboto）
- 标题字号：24-32pt
- 正文：14-16pt

### 4.3 动效
- 页面切换：淡入淡出 200ms
- 评分进度：数字滚动 800ms
- 偏差条：弹簧动画 150ms
- 节拍器心跳：精确 60 BPM 下 1s/拍
